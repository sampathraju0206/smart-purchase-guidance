# Smart Purchase Guidance using Size/Fit Mismatch and Return Risk Prediction for E-Commerce

## Project Overview
This project focuses on reducing e-commerce returns by predicting size/fit mismatch and return risk before purchase. The system uses product, customer review, rating, and fit-related information to identify products that are more likely to be returned due to sizing or expectation mismatch.

## Problem Statement
E-commerce platforms, especially in fashion and footwear categories, face a high number of returns because customers receive products that do not match their size expectations, fit preferences, or product understanding. These returns increase logistics costs and reduce customer satisfaction.

## Objective
The objective of this project is to build a machine learning solution that:
- predicts return risk for a product purchase
- identifies size/fit mismatch patterns from customer feedback
- helps customers make better purchase decisions
- helps e-commerce platforms reduce avoidable returns

## Project Goals
- Understand the relationship between reviews, ratings, size feedback, and returns
- Perform exploratory data analysis on fit/size-related product data
- Build features related to fit, review sentiment, and product behavior
- Train ML models for return risk prediction
- Generate insights that can support smart purchase guidance
